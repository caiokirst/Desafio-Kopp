# Exercício 1 

O exercício 1 é um programa Java simples que gera uma frase formatada com valores de uma lista de números fornecida.

## Como executar

Certifique-se de ter o Java instalado em sua máquina.

1. Clone o repositório para o seu computador ou faça o download dos arquivos do código-fonte.

2. Navegue até o diretório do projeto:

3. Compile o código

4. Execute o programa:

## Exemplo de Uso

O programa já possui uma lista de números predefinida. Ao ser executado, ele gerará uma frase formatada com os valores da lista e mostrará o resultado no console.

Exemplo de saída: "Remessa gerada: 1 cujo valor R$ 88,00, 2 cujo valor R$ 130,00, 3 cujo valor R$ 54,90, 4 cujo valor R$ 293,30, 5 cujo valor R$ 44,80. Total = R$ 611,00"

## Personalização

Se desejar utilizar uma lista de números diferente, basta modificar a lista na classe `App` antes de executar o programa:

1. List<Double> numeros = Arrays.asList(100.00, 50.00, 75.50, 200.20);
   Exemplo de saída: "Remessa gerada: 1 cujo valor R$ 100,00, 2 cujo valor R$ 50,00, 3 cujo valor R$ 75,50, 4 cujo valor R$ 200,20. Total = R$ 425,70"

2. List<Double> numeros = Arrays.asList(729.25, 89.50, 22.90, 15.90, 7.75, 180.30);
   Exemplo de saída: "Remessa gerada: 1 cujo valor R$ 729,25, 2 cujo valor R$ 89,50, 3 cujo valor R$ 22,90, 4 cujo valor R$ 15,90, 5 cujo valor R$ 7,75, 6 cujo valor R$ 180,30. Total = R$ 1045,60"
